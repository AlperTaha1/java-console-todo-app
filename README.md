# Java Console Task Management System

A robust, console-based Task Management application built with **Java** and **MySQL**. This project demonstrates core Object-Oriented Programming (OOP) principles, Layered Architecture, and JDBC database integration.

## Features

* **Task Management:** Add, list, complete, and delete tasks.
* **Database Persistence:** All data is stored in a MySQL database, ensuring data isn't lost when the application closes.
* **Layered Architecture:** Clean separation of concerns using Service, Repository, and Model layers.
* **Custom Exception Handling:** Robust error handling with custom exceptions (e.g., `TaskNotFoundException`).
* **Maven Support:** Dependency management handled via Maven.

## Tech Stack

* **Language:** Java (JDK 17+)
* **Database:** MySQL
* **Build Tool:** Maven
* **Database Driver:** JDBC (MySQL Connector/J)
* **IDE:** IntelliJ IDEA
